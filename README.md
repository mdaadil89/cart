# cart
React Shopping Cart

# Getting started
### Requirements

* Node.js
* NPM
* Json-Server 

### Package installation

```bash
npm install
```
### Starting JSON Server

```bash
json-server --watch src/db.json -p 4000
```

 ### Start the React App
 Excute the following command: 
```bash
npm start
```
The application will start automatically in your browser on http://localhost:3000
